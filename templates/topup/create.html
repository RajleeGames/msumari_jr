What happens after save

System behavior for this transaction.

1
Normal stock reduces

The selected new product quantity is deducted from normal branch stock.

2
Used stock is created

The used item received enters used Top-up stock, not normal stock.

3
Addon money is recorded

The addon amount goes only to the Top-up dashboard and stays separate from normal sales.

4
Used item can be sold later

Later, staff can sell the used item at any amount and it still remains inside Top-up reports.

Top-up Tips

Keep entries clean for better reports.

Use clear used item names

Example: “Used TV 24 inch LG” is better than just “TV”.

Accepted value matters

The buying price here becomes the cost base when later calculating used-item profit or loss.

Write useful notes

Add details like remote, cracks, scratches, or screen condition so resale staff can know the item well.
